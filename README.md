# food_waste_app

A new Flutter project.
![Flutter](https://img.shields.io/badge/Flutter-Mobile-blue)
![AWS](https://img.shields.io/badge/AWS-Amplify-orange)

A Flutter-based mobile application designed to reduce food waste by connecting food donors with NGOs, orphanages, and animal shelters.

> Turning surplus food into meaningful impact.

---

## 🚀 Features

- 🔐 Secure Authentication (Login / Signup)
- 🏢 Organization Registration (NGO / Orphanage / Blue Cross)
- 📦 Add Food Listings with detailed metadata
- 🤝 Claim Food from nearby donors
- 📷 Image Upload Support
- 📍 Location Sharing via Google Maps links
- 📊 Report Generation (PDF & Excel)
- 🤖 Built-in Assistant Guide (non-chatbot)

---

## 🛠 Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** AWS Amplify
- **Authentication:** AWS Cognito
- **Storage:** Cloud-based (Amplify Storage)

---

## ⚙️ Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio / VS Code
- Emulator or physical device

### Installation

```bash
git clone https://github.com/chills-7-ds/food-waste-management-app.git
cd food-waste-management-app
flutter pub get
flutter run
```
## 🔐 Note on Backend

- AWS Amplify configuration files are excluded for security reasons.

> To run backend features:

- Configure your own AWS Amplify environment
- OR request access to the original backend setup
